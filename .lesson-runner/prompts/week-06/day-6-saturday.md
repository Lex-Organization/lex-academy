# Lesson 6 (Module 6) — Interview & Quiz

## What You Covered This Week
This week you learned React fundamentals and built the embroidery store as a React application:
- **Lesson 1:** JSX syntax, functional components, props, children, and component tree thinking. Built ProductCard, ProductGrid, Badge, PriceTag, CategoryCard, and StoreBanner components for the embroidery store.
- **Lesson 2:** useState hook, controlled inputs, form state. Built cart state with quantity controls, search/filter input for products, and a custom embroidery order form with live price preview.
- **Lesson 3:** Event handling with TypeScript, conditional rendering patterns, rendering lists with keys. Built the store's filterable product catalog with category tabs, sort options, and a full cart page with inline editing and order summary.
- **Lesson 4:** Component composition, lifting state up, callback props. Built shared cart state between header badge, product catalog, and cart drawer. Added wishlist feature with move-to-cart/move-to-wishlist transfers.
- **Lesson 5:** Build day. Built a complete React embroidery storefront with product browsing, search/filter/sort, cart with order summary, wishlist, custom order form, and all empty states handled.

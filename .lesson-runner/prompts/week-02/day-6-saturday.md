# Lesson 6 (Module 2) — Interview & Quiz

## What You Covered This Week
- **Lesson 1:** DOM manipulation -- the DOM tree concept, querySelector/querySelectorAll, reading element properties (textContent, classList, dataset, parentElement, closest), creating elements with createElement, textContent vs innerHTML security, DocumentFragment for batch rendering. Built dynamic product catalog that renders entirely from a JavaScript data array.
- **Lesson 2:** Events -- addEventListener, event object (target, currentTarget, type), event types (click, input, submit, keydown), preventDefault. Event bubbling, capturing, stopPropagation. Event delegation pattern (single listener on parent with closest()). Built "Add to Cart" with event delegation, cart count badge, and visual button feedback.
- **Lesson 3:** Forms -- HTML form elements (input types, select, textarea, checkbox, radio), fieldset/legend, FormData API. Custom validation with JavaScript, real-time validation (validate on blur, then on input for touched fields), inline error messages. Built contact/custom order form with conditional fields and newsletter signup with checkbox validation.
- **Lesson 4:** Modern JavaScript -- destructuring (arrays, objects, nested, defaults, renaming), spread operator (clone, merge, immutable updates), rest parameters, optional chaining (?.), nullish coalescing (??), template literals for HTML, Map and Set, modern array methods (at, findLast, Object.groupBy, structuredClone). Built sliding cart drawer with add/remove/update quantity, focus trapping, and event delegation.
- **Lesson 5:** Build day -- interactive embroidery store v1 with product filtering by category, sorting (price, name, rating), debounced live search, combined filter+sort+search pipeline, no-results state, active filter indicators, responsive design (320px-1024px), and keyboard navigation throughout.

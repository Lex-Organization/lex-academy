# Lesson 6 (Module 7) — Interview & Quiz

## What You Covered This Week
This week you dove deep into React hooks and enhanced the embroidery store:
- **Lesson 1:** useEffect for side effects, cleanup, and dependency arrays. Fetched products from a mock API with loading/error states, synced cart and wishlist to localStorage, built debounced product search, and a sale countdown timer with auto-refresh.
- **Lesson 2:** useRef for DOM access and mutable values, forwardRef for exposing DOM nodes. Built search input auto-focus with "/" keyboard shortcut, product image zoom/pan, scroll-to-top button, click-outside drawer detection, and a product customizer with text positioning.
- **Lesson 3:** Performance optimization with useMemo, useCallback, and React.memo. Optimized the product list rendering with 150+ products, built a performance dashboard, and ensured theme toggles cause zero ProductCard re-renders.
- **Lesson 4:** Custom hooks for reusable logic. Built useCart, useProducts, useSearch, useWishlist, useLocalStorage, useDebounce, useToggle, useRecentlyViewed, and useProductFilters — a complete hook library for the embroidery store.
- **Lesson 5:** Build day. Enhanced the embroidery storefront with all hooks integrated: API data fetching, debounced search, persistent cart/wishlist, advanced filtering with active filter count, recently viewed products, image zoom, sale countdown, and optimized rendering.

- **Lesson 4:** Route Handlers — `route.ts` with GET/POST/PUT/DELETE, dynamic params, query parameters, headers and cookies, streaming, catalog export — /api/products, /api/cart, /api/orders endpoints
- **Lesson 5:** Built the full-stack embroidery store combining Server Components, Server Actions, caching strategies, and Route Handlers

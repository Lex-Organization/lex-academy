# Lesson 6 (Module 11) — Interview & Quiz

## What You Covered This Week
This week you learned Next.js data patterns and built the full-stack embroidery store:
- **Lesson 1:** Data fetching in Server Components — async components, fetch with caching options, parallel vs sequential fetching, `Promise.all()`, request memoization — server-rendered product listing and detail pages
- **Lesson 2:** Server Actions — `"use server"`, cart mutations (add to cart, update quantity, remove), `revalidatePath`/`revalidateTag`, `useFormStatus`, `useActionState`, progressive enhancement, wishlist
- **Lesson 3:** Caching deep dive — request memoization, data cache, full route cache, ISR with `revalidate`, tag-based invalidation — cached product pages with inventory-triggered revalidation
- **Lesson 4:** Route Handlers — `route.ts` with GET/POST/PUT/DELETE, dynamic params, query parameters, headers and cookies, streaming, catalog export — /api/products, /api/cart, /api/orders endpoints
- **Lesson 5:** Built the full-stack embroidery store combining Server Components, Server Actions, caching strategies, and Route Handlers

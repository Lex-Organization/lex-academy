# Lesson 6 (Module 8) — Interview & Quiz

## What You Covered This Week
This week you learned React patterns and architecture:
- **Lesson 1:** Context API for dependency injection — built theme, auth, notification, and locale context systems with optimized re-render strategies (split contexts, memoized values).
- **Lesson 2:** useReducer for complex state management — built a shopping cart with typed action patterns and a multi-step checkout flow modeled as a state machine.
- **Lesson 3:** Error boundaries for resilient UIs, React.lazy for code splitting, Suspense for declarative loading states. Built an app shell with granular error handling and a Module Dashboard with independently loading/failing panels.
- **Lesson 4:** Compound components, slots pattern, and headless components. Built production-quality Tabs, Accordion, Dropdown, Select, and AlertDialog components with keyboard navigation and ARIA attributes.
- **Lesson 5:** Build day. Built a complete e-commerce storefront combining all patterns: product catalog, cart (useReducer + Context), theme and auth (Context), lazy-loaded pages (Suspense), error boundaries, and compound components throughout.

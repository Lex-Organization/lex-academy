# Lex: Academy

**Learn frontend engineering by building a real project — guided by AI.**

Lex: Academy is an 18-module interactive course that takes you from zero to job-ready frontend engineer. Each lesson is a prompt you copy into any AI assistant (ChatGPT, Claude, Gemini — your choice). The AI becomes your personal tutor: warm, patient, conversational, and always building toward a real product.

## What You Build

An **embroidery e-commerce store** — from a blank HTML file to a full-stack deployed Next.js app with authentication, a database, an admin panel, CI/CD, and AI-powered features.

The store grows with you. Module 1 it's a static HTML page. By Module 14 it's a production-grade full-stack application.

## What You Learn

| Modules | Topics |
|---------|--------|
| 1–3 | HTML, CSS, JavaScript from the ground up |
| 4–5 | TypeScript — types, generics, real-world migration |
| 6–9 | React — components, hooks, state, routing, testing |
| 10–12 | Next.js — App Router, Server Components, auth, databases |
| 13–14 | Full-Stack Project — architecture, sprint planning, CI/CD |
| 15–16 | Tailwind CSS, shadcn/ui, Playwright, accessibility |

## How It Works

1. Open the **Lex: Academy** panel in VS Code
2. Pick a lesson and click **Copy prompt**
3. Paste into any AI assistant
4. Learn by doing — the AI teaches one concept at a time, interactively
5. Check off items as you go — progress is saved across sessions

Each module has **5 lessons + 1 review day** with a mock interview and quiz.

## Features

- **108 lessons** — carefully crafted prompts covering the full frontend stack
- **LLM-agnostic** — works with any AI assistant, no vendor lock-in
- **Progress tracking** — checkboxes persist across sessions
- **"Go deeper" mode** — optional deep dives for curious learners
- **Real-world practices** — git, tickets, code review, sprint planning, CI/CD
- **One continuous project** — every lesson builds on the last

## Philosophy

**Fundamentals over libraries.** You learn `useReducer` before you hear about Redux. You write `fetch` before you touch a data library. You understand the platform first — then the tools make sense.

**Conversational, not lecture.** The AI teaches one thing, asks you to try it, then waits. No walls of text. No "now implement the entire component." Small bites, real feedback, actual understanding.

**No assumed knowledge.** Module 1 starts from "what is HTML." Module 18 ends with a deployed portfolio and mock interviews. The gap is real — and the course is designed to bridge it.

# Changelog

## 1.0.3

- Prepared the extension for Marketplace publishing with a PNG listing icon, gallery banner metadata, stronger keywords, and clearer extension metadata.
- Rewrote the Marketplace README around audience, workflow, curriculum, privacy, requirements, and install/update behavior.
- Aligned the root README with the Marketplace positioning while keeping developer setup instructions.
- Added `.vsix` files to the extension packaging ignore list so generated packages are never nested into future packages.

## 1.0.2

- Added student support rails to all 108 lessons: starting state, expected outcome, acceptance criteria, stuck prompts, glossary practice, portfolio evidence, and AI pairing guardrails.
- Refined Module 17 and 18 language around AI-assisted engineering so the course teaches pairing with AI rather than building an AI harness.
- Removed screenshot requirements from the curriculum and replaced them with demo notes, visual notes, traces, logs, and written walkthroughs.
- Fixed the GitHub release workflow pnpm setup and published the `v1.0.2` GitHub Release.

## 1.0.1

- Prepared student-support course content and bumped the extension version.
- This tag was superseded by `1.0.2` after a CI workflow configuration issue.

## 1.0.0
